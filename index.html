Github Pull Request jQuery Plugin

Github doesn't make it all that easy to get a list of your pull requests but you can get the information from their API in a slightly roundabout way. This plugin makes listing your pull requests easy.

Usage

  1. Add the plugin to your document's head 
    <script src="pull-request-widget.js"></script>
  2. Add an element with your username 
    <div id="pull-requests" data-github-user="lenniboy"></div>
  3. Instantiate the widget 
    $(document).ready(function(){
  $("#pull-requests").pullRequests();
});

Caching

Since each browser has only 5000 API calls per day and this widget needs to make a lot of requests, the pull request data is cached in localStorage for 30 minutes.
